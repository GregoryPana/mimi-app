# UI_STYLE_GUIDE.md
## Our Love Story — Pastel Modern Romantic UI (for Mimi)

### Design Goal
A modern, minimal, premium romantic app with soft pastel colors.
The UI should feel warm and beautiful, not childish.

---

## 1) Visual Mood
- Soft
- Clean
- Romantic
- Calm + cozy
- Elegant

Avoid:
- Neon colors
- Harsh contrast
- Heavy red everywhere
- Overly playful fonts

---

## 2) Pastel Color Palette (Recommended)
### Backgrounds
- App Background: #FFF7FB (pink-white)
- Secondary Background: #F6FAFF (blue-white)
- Card Background: #FFFFFF

### Accents
- Pastel Pink: #F8BBD0
- Pastel Lavender: #DCC6FF
- Pastel Peach: #FFD6C0
- Pastel Blue: #BFE6FF
- Pastel Mint: #CFF7E6

### Text
- Primary Text: #2B2B2B
- Secondary Text: #6B6B6B
- Muted Text: #9A9A9A

### Borders / Dividers
- Border: #F0E6EF
- Divider: #F3EDF2

### States
- Success: #8DE0C2
- Locked/Disabled: #D9D9D9

---

## 3) Typography
Use a modern font:
- Poppins (recommended) OR Inter OR Nunito

Sizing:
- Screen titles: 26–30
- Card titles: 18–20
- Body text: 14–16
- Small text: 12–13

Rules:
- Use soft wording: “Baby”, “Mimi”, “My love”
- Keep emojis minimal but tasteful (❤️ 💌 🎁)

---

## 4) Layout & Spacing
- Screen padding: 16–20
- Card padding: 16
- Vertical spacing between cards: 12–16
- Use whitespace for a premium feel

---

## 5) Cards
Style:
- Radius: 20–24
- Soft shadow (low opacity)
- Optional subtle gradient header strip:
  - Pink → Lavender
  - Peach → Pink
  - Mint → Blue

Card content:
- Icon (pastel)
- Title
- Subtitle
- Optional status pill (Locked / Completed / Progress)

---

## 6) Buttons
Primary button:
- Rounded radius 18–22
- Pastel pink or lavender fill
- Gentle pressed animation (scale 0.98)

Secondary button:
- White fill
- Pastel border
- Accent text color

Disabled:
- Gray fill
- No shadow

---

## 7) Animations (Must Feel Romantic)
Use subtle animations:
- Fade + slide up transitions
- Hero animation for gallery image open
- Soft “unlock” animation for Surprise Gift + Valentine’s unlock

Avoid:
- Aggressive bounces
- Fast flashing effects

---

## 8) Screen Notes
### Splash Screen
- Gradient background (pink → lavender)
- Heart icon fade-in
- “Enter” button

### Passcode Screen
- Minimal keypad
- Soft hint text
- Cute wrong-code feedback

### Home Screen
Cards:
1) Memory Gallery
2) Our Timeline
3) Surprise Gift (locked until complete)
4) Valentine’s Mode (countdown + letters)

### Timeline Screen
- Vertical timeline feel
- Each item inside a rounded card
- End button: “I finished our story ❤️”

### Gallery Screen
- Rounded thumbnails grid
- Fullscreen viewer with caption
- Hero transition

### Surprise Gift Screen
Locked state:
- Lock icon + checklist
Unlocked state:
- Title: “Happy 4 Years Together ❤️”
- Big romantic message card

### Valentine’s Mode
Before Feb 14:
- Countdown card
- Today’s love letter card
On Feb 14:
- Unlock + show mini-game placeholder
- Show vouchers list

---

## 9) Final Polish Checklist
- App feels premium
- No clutter
- Pastel theme consistent everywhere
- Unlock logic is correct
- Progress states persist after closing app
